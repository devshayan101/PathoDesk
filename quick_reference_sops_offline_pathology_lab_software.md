# QUICK REFERENCE SOPs (1-PAGE)
## Offline Pathology Lab Software

These SOPs are designed to be **printed, laminated, and placed near workstations**.

---

## SOP 1 — Patient Registration (Reception)

1. Verify patient name spelling
2. Confirm DOB and gender
3. Save patient
4. Create order
5. Print labels

❌ Do not create duplicate patients

---

## SOP 2 — Sample Collection & Accession (Technician)

1. Match label with patient
2. Check container & volume
3. Mark sample received
4. Reject if compromised

❌ Never handwrite sample IDs

---

## SOP 3 — Result Entry (Technician)

1. Open pending sample
2. Enter values carefully
3. Review flags
4. Save draft (F5)
5. Submit for verification (F9)

❌ Do not interpret results

---

## SOP 4 — Critical Value Handling

1. Read alert
2. Add mandatory comment
3. Inform supervisor/pathologist
4. Submit

❌ Never ignore critical alerts

---

## SOP 5 — Report Verification (Pathologist)

1. Review all results
2. Check critical & delta flags
3. Add interpretation
4. Preview report
5. Finalize

❌ Finalized reports cannot be edited

---

## SOP 6 — Backup Check (Admin)

1. Check backup status
2. Verify last backup date
3. Ensure external copy weekly

❌ Never ignore backup failures

---
