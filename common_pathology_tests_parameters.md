# Common Pathology Tests & Parameters (Clinically Re‑validated)
## Standard Adult Reference Ranges – Offline LIS Default Dataset

---

> **Clinical disclaimer (important):**  
> The reference ranges below are **widely accepted adult ranges** compiled from standard pathology textbooks, NABL‑aligned lab manuals, and large Indian diagnostic lab practices.  
> **Final authority remains with the reporting lab.** Your software must allow per‑lab, per‑age, per‑gender, per‑method overrides via Test Master versioning.
